# Homebrew Tap — MakMak

Homebrew formulae and casks for [MakMak](https://github.com/dutkiewiczmaciej) tools.

## Install

```bash
brew tap dutkiewiczmaciej/tap
brew install --cask maklock
```

## Available Casks

| Cask | Description |
|------|-------------|
| `maklock` | Lock apps with Touch ID, Apple Watch, or password |
